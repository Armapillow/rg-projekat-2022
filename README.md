# Projekat iz grafike

## Uputstvo
1. `git clone https://github.com/Armapillow/rg-projekat-2022`
2. CLion -> Open -> path/to/my/project_base
3. Main se nalazi u src/main.cpp
4. ALT+SHIFT+F10 -> project_base -> run
